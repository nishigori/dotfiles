Dotfiles
========

```sh
$ git clone --recursive https://github.com/nishigori/dotfiles.git && cd dotfiles
$ make
$ make install
$ git remote set-url origin ssh://git@github.com:nishigori/dotfiles.git
```

## Manually Setup

### MacOS

```sh
xcode-select --install
sudo xcodebuild -license accept
```
