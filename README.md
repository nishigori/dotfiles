Dotfiles
========

```sh
$ git clone --recursive https://github.com/nishigori/dotfiles.git && cd dotfiles
$ make
$ make install
```

## Manually Setup

### MacOS

```sh
xcode-select --install
sudo xcodebuild -license accept
```

* Switch Ctrl <-> Caps Lock: `システム環境設定 > キーボード > 装飾キー`
